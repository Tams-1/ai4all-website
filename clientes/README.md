# Client Pages

Each client gets a subdirectory with:
- index.html — institutional page
- privacy-policy.html — LGPD-compliant privacy policy (required by Meta/WhatsApp)
- terms.html — terms of use

Required by Meta for WhatsApp Business API compliance.
